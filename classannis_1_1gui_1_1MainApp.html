List<File> annis.gui.MainApp.getAllConfigLocations ( String  configFile,
WebApplicationContext  context 
)
protected

Given an configuration file name (might include directory) this function returns all locations for this file in the "ANNIS configuration system".

The files in the result list do not necessarily exist.

These locations are the

  • base installation: WEB-INF/conf/ folder of the deployment.
  • global configuration: $ANNIS_CFG environment variable value or /etc/annis/ if not set
  • user configuration: ~/.annis/
    Parameters
    configFileThe file path of the configuration file relative to the base config folder.
    contextThe context is needed to get he base installation folder
    Returns
    list of files or directories in the order in which they should be processed (most important is last)
