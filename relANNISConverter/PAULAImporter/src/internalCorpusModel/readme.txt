Was behinaltet das Paket internalKorpusModel (KM)?
=====================================

In diesem Paket befinden sich alle Klassen die benötigt werden um das interne Korpus Model (IKM) darzustellen. 
Die hier enthaltenen Klassen bilden die Darstellung des Internen Modells, davon abgeleitete Klassen realisieren die Abbildung auf ein reales 
Datanbankmodell.
